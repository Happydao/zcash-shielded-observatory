Source / Methodology

Data Source and Method

Data source

ZcashInfo API primary.

Method

Current value pools are read from public Zcash value-pool accounting. Patch balance is read from block 3,363,426. All values are converted from zatoshis to ZEC.

Update model

The static dataset is refreshed by the update script and served directly by GitHub Pages.

Limitations

Scope of Public Data

This dashboard tracks public value-pool accounting and post-patch movement.

Private Orchard notes are not publicly visible and cannot be independently audited through value-pool balances alone.
